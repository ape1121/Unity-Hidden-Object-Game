# Search It - Vertical Slice

Search It is a Unity hidden-object vertical slice focused on clean architecture and rapid AI-assisted iteration.

Targeted Resolution : 1284 x 2778  
Unity Version : 6000.3.6f1

## Goals for This Slice

- General application structure and architecture
- Game session management (start, pause, end)
- In-game mechanics (item finding, camera zoom/pan)
- Coin system based on elapsed time
- UI screen management (menus, transitions, popups)
- AI-generated art asset integration
